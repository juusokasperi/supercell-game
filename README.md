# Endless Runner
My submission for Supercell's coding challenge at Hive Helsinki 2026.

## Prerequisites

CMake version 3.22.1 or later.

## Build & Run

Run `cmake -B build` and `cmake--build build`. Your executable is in `./build/bin/`.

## Game Instructions:

Follow instructions in main menu (press `ENTER` to start).

### Controls:

```
LEFT SHIFT: Accelerate speed while held
Z: Decelerate speed while held
SPACEBAR: Jump - holding makes you jump higher.

ESC: Pause menu for quitting and checking controls.
```
